# CMSN (Commission)

**A luxury-neutral athleisure brand built with Next.js, TypeScript, and AI-assisted development.**

Live at [cmsn-silk.vercel.app](https://cmsn-silk.vercel.app)

## What This Is

CMSN (Commission) is a luxury-neutral athleisure brand — minimal design, neutral palette, elevated everyday wear. This repository is the web presence and commerce foundation.

Built in collaboration with Cursor AI, demonstrating how a non-engineer can ship production-ready Next.js applications using AI-assisted development workflows.

Commission represents the idea that everything worth having is earned. The brand is built on that principle.

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Vercel (Production)
- Cursor AI
